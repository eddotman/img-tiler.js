Installation

  1. Make sure you have jQuery included.
  2. Include img-tiler.js.
  3. Include the following in your HTML:
    <img id="img1" src="img/YOUR_IMG_HERE.jpg" style="display:none">
    <canvas id="c1" width="xxx" height="yyy" ></canvas>
    <canvas id="c2" width="xxx" height="yyy"></canvas>
    <input type="text" id="rows">
    <input type="text" id="cols">
    Where xxx and yyy are the appropriate width/height.
  4. Add CSS styles and HTML as necessary.

img-tiler.js will render the unsliced image in the canvas with ID "c1," and the sliced image in the canvas with ID "c2". Slicing is done using the parameters from the inputs with IDs "rows" and "cols."
